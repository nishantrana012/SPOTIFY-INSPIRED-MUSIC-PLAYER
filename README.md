# 🎵 Spotify Clone (Music Player)

A responsive, web-based music player interface inspired by Spotify's design. This project demonstrates front-end development skills using vanilla web technologies to create a functional audio playback experience.

## ✨ Features

* **Playback Controls:** Play, pause, next, and previous functionality.
* **Progress Bar:** Clickable seek bar that updates in real-time.
* **Dynamic UI:** Album art and song titles change based on the current track.
* **Responsive Design:** Optimized for desktop and mobile viewing.
* **Spotify Aesthetic:** Dark theme, hover effects, and modern styling.

## 🛠️ Technologies Used

* **HTML5** - Structure and layout
* **CSS3** - Flexbox, CSS Grid, and custom animations
* **JavaScript (Vanilla)** - DOM manipulation and `Audio()` API logic

## 📂 Project Structure

```bash

└── img/             # Folder containing all the images and logos

## 📂 How to Run
1.  Clone the repository:
    ```bash
    git clone https://github.com/nishantrana012/SPOTIFY-INSPIRED-MUSIC-PLAYER.git
    ```
2.  Navigate to the project folder.
3.  Open `index.html` in your browser.
